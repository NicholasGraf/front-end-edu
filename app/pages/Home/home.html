Application Core Modules

We now have some core modules that allow us to start build a Single Page Application:
~/app/assets/js/core/

  • Router
  • Events Module
  • Components Module

Web Components

I have started out SPA with some basic Web Components:
~/app/assets/js/components/

  • App (Application wrapper)
  • PageContainer (Page content wrapper)
  • Navigation & NavItem (For changing views)
  • BaseElement (A root class for other web components)
  • BasePage (A root class for page specific JavaScript)

Application Configuration

I have saved some of our application settings in a configuration file:
~/app/config.js:

  • Application Name
  • Root Selector
  • Page Selector
  • Routes
  • Event Name Strings

Application Global Object

You also should note that we create an Application Global Object (~/app/assets/js/index.js).

It is initiated here: ~/app/assets/js/core/init.js

This allows us to track the state of the application as well as inject dependecies that can be used globally.

As usual, feel free to customize and play with this codebase as we step through the develoment process.
